# Advent of Code

This repository contains solutions to the Advent of Code challenges.

## Structure

- **src/**: Contains the source code for the solutions.
  - **main.rs**: The main entry point for the application.
  - **bin/**: Contains individual challenge solutions.
    - **day1.rs**: Solution for Day 1 challenge.
- **inputs/**: Contains input files for the challenges.
  - **day1.txt**: Input for Day 1 challenge.
- **target/**: Contains build artifacts and cached files.

## Usage

To run the solutions, use the following command:

```bash
cargo run --bin day1
```

Make sure to replace `day1` with the appropriate day for other challenges.
